# Kiosco App

App web para kiosco de barrio. Permite a clientes armar un carrito y finalizar el pedido por WhatsApp.

## Stack
- Next.js 14 (App Router)
- Supabase (base de datos y auth)
- Tailwind CSS (estilos)
- Vercel (deploy)

## Reglas
- Código en español (variables, comentarios)
- Componentes simples, sin over-engineering
- Siempre manejar errores y estados de carga
- Mobile-first (la mayoría usa el celular)

## Fases
1. Catálogo + carrito + WhatsApp ← arrancamos acá
2. Panel admin + stock
3. Historial de ventas
4. Mercado Pago
